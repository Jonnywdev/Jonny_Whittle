Leigh Rufc Table Service

In October 2020 the government announced that all pubs will need to do table service from now on. I knew from my previous experience working on the bar how difficult it will be for the staff to cope. Therefore I decided to create a website for table service. I first created a small prototype on adobe XD which I used to convinced the clubs board that a website is what we need. I had no experience with websites then, I had just finished my degree in Graphic Design the most website designing that I had completed was on squarespace ðŸ˜…. I did end up pulling my friend in who had an idea of how to do backend such as php.

I made sure that I kept the website simple so any ages could use it. I decided not to include a payment method on the website to cover myself incase I did something wrong and customer information was leaked. Unfortunatley we did not need the website anymore so we have taken it down.

In summary, what I built was:

- A easy to use website for both staff and customers.

- Online menu for the pub which was easily alterable.

- Room booking system.

I have listed all of the features below.

Technology Used

- HTML

- CSS

- JAVASCRIPT

- PHP

- MYSQL

Features of the customer side
- Unlimited produts available to add.
- Fully Responsive design.
- Page showing the public events, their availability and description which give customers the ability to buy tickets from their phone.
- Easy to use room booking system.
- Email to confirm whether booking has been confirmed or denied.
- Easy booking form to fill out.

Features of the admin side
- Clearance levels used to allow only certain members to see different options.
- Super secret button, gives easy access to the admin side straight from the homepage.
- Automatic refresh on current order page.
- Order card showing clear information about order and easy to delete or complete order.
- Menu with easy access to product and category management, room hire, staffing and way back to the website.
- Easy to view calendar, with colour system to show the diferences between the bookings.
- Form showing all the customers information (only staff with the right clearance can view these forms).

Features of the epos system
- Reporting, helps to see what products are selling and which are not.
- Members discount, if the user is a member at the pub the discount will come straight off after they have submitted their order. (Makes it easier for staff) there is also a tab at the top of the epos system for quick access to the discount for when people are aloud to go to the bar for a drink.
- Designed and laid out to your requirements.
- Fully integrated system.
